# VitalTrack Website

This folder is a standalone static website for the VitalTrack app.

Open `index.html` directly in a browser, or serve the folder with any static server.

## Files

- `index.html` - page markup
- `styles.css` - responsive styling
- `script.js` - mobile navigation, app preview tabs, and small counter animation
- `assets/vitaltrack_logo.png` - copied logo asset

The site intentionally lives outside Flutter's `web` folder so the app build files stay separate from the marketing website.
